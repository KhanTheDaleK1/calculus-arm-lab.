OSCILLOSCOPE
Zoom X squeezes/expands time; Gain Y lifts small waves. Copy CSV → paste into Sheets/Desmos (Time,Voltage columns). For TI-84: paste into TI-Connect CE List Editor (L1=time, L2=V), then plot Stat Plot. TI-Nspire: import CSV into Lists & Spreadsheet, then Data & Stats. Casio: import CSV to STAT via fx-Manager or C-CSV and set Lists.
TONE GENERATOR
Start engine → set frequency/volume/wave → ACTIVATE TONE. Use sine for clean pitch detection, square/saw for harmonics. Stop before exporting to avoid feedback.
SONAR (ECHO)

Emits chirp. Measures reflection time.

-- ms / -- m
Requires loud speaker.
Face a wall, turn up speakers. Press PING. App blanks first 50 ms to ignore self-sound; reads echo delay→distance. If blank, try louder/closer wall or external speaker. Not for calculator export (use scope/spectrum CSV).
SPEED OF SOUND

Detects two sharp claps.

-- ms
-- m/s
Status: Idle
ARM → clap once (loud) → 200 ms lockout → clap again within ~3s. Set DIST to your separation. Result shows Δt and speed. To log on calculators, record the two times manually into lists.

CALCULATOR EXPORT
TI-84 CE: COPY CSV → TI-Connect CE → List Editor → paste into L1/L2 → STAT PLOT line/scatter.
TI-Nspire: COPY CSV → Lists & Spreadsheet (Menu > Data > Import) → plot in Data & Stats.
Casio (fx/Graph): COPY CSV → fx-Manager Plus or C-CSV → load into STAT Lists → draw graph.
Sheets/Desmos: Paste CSV directly; Desmos auto-plots two-column data.
